# శ్రీ విష్ణు సహస్ర నామ స్తోత్రం | Sri Vishnu Sahasranama Stotram

A beautiful, devotional website featuring the **Sri Vishnu Sahasranama Stotram** with:

- ✨ **107 Shlokas** in Telugu script (from authentic Vaidika Vignanam source)
- 📝 **English transliteration** for each verse (for pronunciation & learning)
- 🌐 **Meanings in both Telugu and English**
- 🪔 **Dhyana Shlokas** (meditation verses)
- 🎁 **Phalashruti** (benefits of chanting)
- 🔍 **Search** across names, shlokas, and meanings
- 📌 **Bookmark** your favorite shlokas (saved in browser)
- 🔤 **Font size controls** for readability
- 🎨 **Traditional Indian devotional theme** (saffron, gold, maroon)

## Source

The text is sourced from the **Anushasana Parva of the Mahabharata**, where Bhishma Pitamaha teaches the thousand names of Lord Vishnu to Yudhishthira. The Telugu text follows the **Sarala Telugu** version from [Vaidika Vignanam](https://www.vaidikavignanam.com).

## Tech Stack

- **Next.js 16** with TypeScript
- **Tailwind CSS 4** + **shadcn/ui** components
- **Noto Serif Telugu** font for authentic Telugu rendering
- **Cinzel** display font for headings
- Deployed via **GitHub Pages** using GitHub Actions

## Local Development

```bash
bun install
bun run dev
```

Open http://localhost:3000 in your browser.

## Build for Production

```bash
bun run build
```

The static site will be generated in the `out/` directory.

## Deployment

This site is automatically deployed to GitHub Pages via GitHub Actions. On every push to `main`, the workflow builds the site and publishes it.

**Live site**: https://kartheekbvs.github.io/Vishnu/

## Features

### 📖 Four Main Sections

1. **Stotram** — All 107 main shlokas with Telugu, English, and meanings
2. **1000 Names** — Browseable list of individual divine names
3. **Dhyana** — Meditation verses + Phalashruti (benefits)
4. **Saved** — Your bookmarked shlokas

### 🎨 Design

- Traditional saffron-gold-maroon color palette
- Mandala-inspired decorative elements
- Om symbol motifs
- Responsive mobile-first design
- Beautiful Telugu typography
